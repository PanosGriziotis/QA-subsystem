Εξηγήστε στους φίλους σας γιατί είναι σημαντικό να προστατεύσουν τους εαυτούς τους και άλλους πλένοντας τα χέρια τους, αποφεύγοντας να αγγίζουν το πρόσωπό τους, πάντα βήχα ή φτερνίζοντας στον αγκώνα τους, το μανίκι τους, ή έναν ιστό, και συνεργάζοντας με σωματικά αποστασιοποιητικά μέτρα και περιορισμούς κίνησης όταν καλούνται να το κάνουν αυτό.Ίσως μπορείτε να μοιραστείτε ιδέες για διασκεδαστικές εικονικές δραστηριότητες στις οποίες μπορούν να συμμετάσχουν οι φίλοι σας, και μπορείτε να τους ενθαρρύνετε να τα κάνουν μαζί σας ή με άλλους φίλους. Για παράδειγμα, μπορείτε να τους ενθαρρύνετε να ενταχθούν στην εκστρατεία YouthAnstandCOVID19 που στοχεύει να βοηθήσει να διδάξουν νέους ανθρώπους σε όλο τον κόσμο σχετικά με το COVID-19 και τι μπορούν να κάνουν για να κρατήσουν τους φίλους, τις οικογένειες και τις κοινότητές τους ασφαλείς.
